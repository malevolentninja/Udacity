### 1. Numbers 

* Expressions 
Determine what each expression evaluates to using the console:
```sh
2 + 10 - 19 + 4 - 90 + 1 = -92

-20 + -19 - (-10) - (-1) + 24 = -4

(10/5) * 4 - 20 = -12

4096 % 12 = 4
```

* Comparing Numbers 

Determine what each expression evaluates to using the console:
```sh
43 > 47 = false

12 == 17 = false

3 <= 3 = true

1 != 0 = true
```
